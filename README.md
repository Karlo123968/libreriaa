Libreria


Este repositorio contiene un sitio web informativo cuyo tema es una biblioteca, desarrollado como parte de una práctica académica de HTML y CSS.

El sitio parte de una página principal que incluye una imagen utilizada como mapa de imagen, permitiendo al usuario seleccionar distintas áreas de la biblioteca. Cada área dirige a una página interna del sitio, sin utilizar enlaces externos.

El proyecto cuenta con cinco áreas principales: área de lectura, área de cómputo, área infantil, área de préstamo y área de estudio. Cada una tiene su propia página con información descriptiva organizada en varios párrafos, haciendo uso de etiquetas generales y embebidas de HTML.

Además, se utiliza una hoja de estilos CSS externa para dar formato visual al sitio, mejorando la presentación, legibilidad y estructura de las páginas.

Este repositorio tiene fines educativos y demuestra el uso de:

Estructura básica de HTML

Mapas de imagen (map y area)

Enlaces internos

Etiquetas embebidas

Estilos externos con CSS
